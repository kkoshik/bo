# Максимально простой шаблон

Шаблон сделан максимально близко к примеру из архива: простые PHP-страницы, cookies, PDO, один CSS и простой слайдер через `innerHTML`.

## Файлы

- `index.php` - авторизация.
- `reg.php` - регистрация.
- `db.php` - подключение к базе данных.
- `forms/auth_form.php` - форма авторизации.
- `forms/reg_form.php` - форма регистрации.
- `home.php` - личный кабинет, заявки, отзывы, слайдер.
- `formirovanie.php` - создание заявки.
- `admin.php` - просмотр заявок и смена статуса.
- `logout.php` - выход.
- `slider.js` - слайдер как в примере.
- `css/global.css` - все стили.
- `database.sql` - база данных.

## Что заменить в коде

Во всех PHP-файлах заменить:

- `НАЗВАНИЕ_САЙТА` на название варианта.
- `ОПИСАНИЕ_САЙТА` на короткое описание.
- в `formirovanie.php` заменить `ВАРИАНТ_1`, `ВАРИАНТ_2`, `ВАРИАНТ_3`, `ВАРИАНТ_4`.
- в `admin.php` заменить `СТАТУС_2`, `СТАТУС_3`.
- в папке `img` заменить `logo.jpg`, `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`.

Если в варианте только 3 пункта выбора, четвертый `option` можно удалить.
Если дата рождения не нужна, в `reg.php` можно удалить строку с `birthday`, но проще оставить.

## В1 Водить.РФ

- `НАЗВАНИЕ_САЙТА`: `Водить.РФ`
- `ОПИСАНИЕ_САЙТА`: запись на курсы вождения речного транспорта.
- варианты: `Катер`, `Круизный лайнер`, `Яхта`
- статусы: `Идет обучение`, `Обучение завершено`
- картинки: речной транспорт.

## В2 Конференции.РФ

- `НАЗВАНИЕ_САЙТА`: `Конференции.РФ`
- `ОПИСАНИЕ_САЙТА`: бронирование помещений для конференций.
- варианты: `Аудитория`, `Коворкинг`, `Кинозал`
- статусы: `Мероприятие назначено`, `Мероприятие завершено`
- картинки: конференции и помещения.

## В3 Пассажирам.РФ

- `НАЗВАНИЕ_САЙТА`: `Пассажирам.РФ`
- `ОПИСАНИЕ_САЙТА`: запись на курсы водителей пассажирского транспорта.
- варианты: `Автобус`, `Электробус`, `Трамвай`
- статусы: `Идет обучение`, `Обучение завершено`
- картинки: городской транспорт.

## В4 Банкетам.Нет

- `НАЗВАНИЕ_САЙТА`: `Банкетам.Нет`
- `ОПИСАНИЕ_САЙТА`: бронирование помещений для банкетов.
- варианты: `Зал`, `Ресторан`, `Летняя веранда`, `Закрытая веранда`
- статусы: `Банкет назначен`, `Банкет завершен`
- картинки: банкетные помещения.

## Запуск

1. Положить папку в `htdocs`.
2. Импортировать `database.sql` в phpMyAdmin.
3. Открыть `index.php`.

Админ: `Admin26` / `Demo20`.
